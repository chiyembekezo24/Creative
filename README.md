# Creative

A modern one-page portfolio and agency website for Creative, built with HTML and CSS and deployed as a static site.

## Features
- Responsive landing page
- Services, process, and contact sections
- Working contact form with anti-spam check
- Social links for LinkedIn and GitHub

## Deployment
This site is configured for Render as a static site.

### Render setup
- Create a new Static Site on Render
- Connect this GitHub repository
- Set the publish directory to `.`
- Deploy

## Local preview
Open [index.html](index.html) in a browser or serve the folder with a simple local server.
